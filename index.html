Github Command List

haghaa

  • git clone + (Link of the repository)

    Clones the files in the link added

  • git add (file name OR .)

    adds the change files

  • git status

    Check status

  • git commit

    Commits changes to the repository

  • git push

    Uploads the files in the repository

  • git pull

    pulls whatever is the latest

  • git branch

    checks your current branch location

  • git branch + (branch name)

    creates a new branch name

  • git checkout + (branch name)

    changes your current branch
